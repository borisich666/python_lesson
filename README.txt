Описание проекта: Генератор комбинаций карт
Цель проекта
Создание программы для генерации всех возможных комбинаций из заданного количества карт в стандартной колоде. Программа позволяет пользователю указать количество карт, и на основе этого генерирует все уникальные комбинации, выводя их на экран или сохраняя в файл для последующего использования.

Шаги реализации
Создание колоды:
Программа формирует стандартную колоду из 52 карт, состоящую из четырех мастей: пики (♠), черви (♥), бубны (♦) и трефы (♣). Каждая масть включает ранги от 2 до 10, а также валет (J), даму (Q), короля (K) и туза (A).
Генерация комбинаций:
С использованием модуля itertools программа генерирует все возможные комбинации из заданного количества карт (от 1 до 52). Каждая комбинация представляется в виде кортежа, состоящего из уникальных карт.
Вывод результатов:
После генерации комбинаций программа предлагает сохранить их в текстовом файле для дальнейшего использования или выводит на экран. Каждый набор комбинаций записывается в файл построчно.
Структура программы
Программа состоит из трех основных функций, а также основной функции main(), которая управляет пользовательским вводом и общим процессом:

create_deck(): Генерирует и возвращает список всех карт в колоде.
generate_combinations(deck, num_cards): Принимает колоду и количество карт, возвращает список всех комбинаций.
save_card(combinations, filename): Сохраняет комбинации в указанный текстовый файл.
read(filename): Читает содержимое файла и возвращает его для отображения пользователю.
Интерактивность
Программа запрашивает у пользователя количество карт для генерации комбинаций и проводит валидацию ввода, гарантируя, что пользователь введет корректное значение в диапазоне от 1 до 52. В случае некорректного ввода, выводится соответствующее сообщение, и программа запрашивает ввод повторно.
